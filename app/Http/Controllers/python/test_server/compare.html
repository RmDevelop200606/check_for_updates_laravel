法事のお席やお身内だけの会席、お祝いの華やかなお席などに
神戸市花隈町の料亭旅館豊福の仕出し料理を是非ご利用下さい。
おもてなしの心をお伝えするお料理は、集う方々にほんのりとした温もりと安らぎを与えます。
食材から吟味された瀬戸内の恵みを、お弁当からコースまで取り揃えております。
法要なのか、お祝いのお席なのか、洋食なのか。その用途に応じてお食事をご用意させて頂きます。
瀬戸内の恵みをはじめ、季節に合わせたお料理をお出ししております。お好みなどありましたらお気軽にご相談下さい。
配達も行っておりますが、店内でお召し上がり頂く事も可能です。エレベーター完備で高齢者や車椅子の方でも大丈夫。

歯が悪くなってきた方にはお惣菜を細かく切るなど召し上がる方の健康や状態を考えて仕出しを調理致します。
